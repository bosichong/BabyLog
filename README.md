
### BabyLog

岁月如风，唯有此忆。 任凭时光匆匆，记录点点滴滴。 ---- 爸爸

BabyLog是用来记录孩子成长过程的日记，她包括：日记，语言，认知等，还包括身高，体重，并使用图表展示。
